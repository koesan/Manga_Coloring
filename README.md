# Google Colab'da GAN Based Architectures for Manga Colorization

In this project, two different GAN based architectures were used to colorize black and white manga images: Pix2PixHD and CycleGAN.
Bu projede, siyah-beyaz manga resimlerini renkli hale getirmek için iki ayrı GAN türevi mimari kullanılmıştır: Pix2PixHD ve CycleGAN.

In the first project, the Pix2PixHD model was used to work with paired black-and-white and colored manga image pairs.
İlk projede, Pix2PixHD modeli kullanılarak eşleşmiş siyah-beyaz ve renkli manga resim çiftleri üzerinde çalışılmıştır.

Pix2PixHD is a GAN model developed by NVIDIA to provide high-quality image transformations.
Pix2PixHD, yüksek kaliteli görüntü dönüşümleri sağlamak üzere NVIDIA tarafından geliştirilmiş bir GAN modelidir.

This model is trained with supervised learning to produce realistically and accurately colored manga images.
Bu model, denetimli öğrenme ile eğitilerek gerçekçi ve doğru bir şekilde renklendirilmiş manga resimleri üretmeyi hedeflemektedir.

In the second project, the CycleGAN model was used.
İkinci projede ise CycleGAN modeli kullanılmıştır.

CycleGAN works with unpaired data pairs to perform transformation between two different domains (black-and-white and colored manga images).
CycleGAN, eşleşmemiş veri çiftleriyle çalışarak iki farklı domain (siyah-beyaz ve renkli manga resimleri) arasında dönüşüm gerçekleştirir.

This model makes color transfer possible with unsupervised learning between different data sets.
Bu model, denetimsiz öğrenme sayesinde farklı veri setleri ile renk transferi yapmayı mümkün kılar.
